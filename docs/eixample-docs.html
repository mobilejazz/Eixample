Eixample

The Eixample (pronounced ey-sham-pla) is a district in Barcelona, characterized by it's strict grid system of long straight streets and wide avenues. Inspired by our city, this CSS flexbox grid is made with love by Mobile Jazz.

Technologies

Eixample is built using Compass, if you wish to modify the default grid, you'll need it installed locally to recompile the styles.

Breakpoints

Eixample is a mobile-first responsive grid, which uses four media queries to create layouts for five screen widths xs sm md lg and xl. These media queries are based on minimum widths and scale up.

// Extra small devices (xs)
// 0px +
// No media query

// Small devices (sm)
// 600px +
@media ( min-width: $breakpoint-sm ) {
}

// Medium devices (md)
// 900px +
@media ( min-width: $breakpoint-md ) {
}

// Large devices (lg)
// 1200px +
@media ( min-width: $breakpoint-lg ) {
}

// Extra large devices (xl)
// 1500px +
@media ( min-width: $breakpoint-xl ) {
}

The width of these break points can be customized in sass/_settings.scss

Containers

Containers are used to constrain the width and centralize your content. They are required and set nessecary spacing.

Basic Container

The default container will expand upto 1200px wide before centralizing on larger screens. You can customize this width by changing the value of the Sass variable $container-max-width in sass/_settings.scss.

To create a container, use the class .container.

<div class="container">
</div>

There are two additional modifier classes that can be used to change a container's style. You can use these independently, or combine them.

Rows

Rows are required to group columns and should be direct children of containers.

Basic Row

Rows have a fluid width and will expand upto any size. They feature negative margins to ensure proper placement for both ltr and rtl languages.

To create a row, use the class .row.

Column Widths

Eixample has a traditional 12 column layout, available for each of the five breakpoints. Use the base class .column with a combination of sizes classes: .xs-* .sm-* .md-* .lg-* .xl-* combined with a column span between *-1 and *-12.

Widths don't have to be set for each breakpoint. If no width is set for .xl-* the width for .lg-* will be used instead. This fallback goes through .md-* .sm-* and finally .xs-* 

<div class="container">
    <div class="row">
        <div class="column xs-12 sm-6 md-3 lg-4 xl-2">
        </div>

        <div class="column xs-12 sm-6 md-3 lg-2 xl-4">
        </div>

        <div class="column xs-6 sm-9 md-3 lg-6">
        </div>

        <div class="column xs-6 sm-3 md-3 lg-12">
        </div>
    </div>
</div>

Automatic Column Widths

Column widths are entirely optional. If you don't specify a width, the column will size itself according to it's content.

However, you can also enforce an automatic width for specific breakpoints by setting the size class with *-auto. 

<div class="container">
    <div class="row">
        <div class="column xs-12 sm-6 md-auto">
        </div>

        <div class="column xs-12 sm-auto md-6">
        </div>
    </div>
</div>

Display Classes

        </div>

        </div>
